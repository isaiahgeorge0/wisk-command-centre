# Goals — Feature Specification

## Current State

What is built and working today.

- Goals grouped active/paused/completed+archived
- Fields: title, category, target, current, unit, deadline, status
- Progress bar with colour banding
- Quick +/- controls
- Progress popover
- Content goal linking with published post count

## Gaps and Missing Features

What is missing or underdeveloped, with a priority rating (High/Medium/Low) per item.

- Goal filtering by category/status — **Medium**
- Progress history over time — **Medium**
- Goal to project linking — **Medium**
- Milestones within goals — **Medium**
- Weekly check-in prompt — **Low**
- Goal templates — **Low**

## Planned Additions (Phase 2)

Features committed to building before Phase 3 begins.

- Goal to project linking
- Progress history

## Future Considerations (Phase 3+)

Features that are on the radar but not yet committed.

- Filtering
- Milestones within goals
- Weekly prompts
- Templates

## Technical Notes

Any important technical context, constraints, or decisions relevant to this section.

- Progress colour banding: 0–33% coral, 34–66% amber, 67–99% teal, 100% green
- Same thresholds used on project task progress bars
